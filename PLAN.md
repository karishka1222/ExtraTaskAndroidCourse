# PLAN.md — Декомпозиция проекта "MVP магазина приложений"

## Описание проекта
Цель: разработать минимально жизнеспособную версию (MVP) Android-магазина приложений.

Функционал MVP:
- список приложений;
- карточка приложения;
- установка приложения (имитация);
- удаление приложения (имитация).

---

# Эпики

1. **Epic 1 — Инициализация проекта и архитектура**
2. **Epic 2 — Экран списка приложений**
3. **Epic 3 — Экран карточки приложения**
4. **Epic 4 — Логика установки и удаления приложений**
5. **Epic 5 — Тестирование и стабильность**
6. **Epic 6 — CI/CD + документация**

Каждый эпик ≤ 2 недель (обычно 3–7 дней).
